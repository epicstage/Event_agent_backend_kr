# APEX Industry Glossary - 주요 용어 발췌

## 소스
- http://mmiresource.hotelpros.biz/doc-folder/Guest_services/Glossary_CIC.pdf
- Convention Industry Council APEX INDUSTRY GLOSSARY (Updated August 10, 2005)

## 핵심 용어 (A-C)

### A
| 용어 | 정의 |
|------|------|
| **APEX** | Accepted Practices Exchange. CIC가 관리하는 업계 표준 이니셔티브 |
| **Accessibility** | 1) 목적지 접근성 2) 장애인 접근 가능성 (ADA) |
| **Accommodation** | 1) 좌석, 침대, 객실, 서비스 2) ADA 하에서 장애인 참여를 위한 조치 |
| **Accompanying Person** | 참석자의 게스트 또는 배우자 |
| **Attrition** | 실제 객실 사용과 계약 수량의 차이. 일정 범위 내에서 허용 |
| **Attrition Clause** | 최소 의무를 충족하지 못할 경우 발생하는 비용/수수료 조항 |

### B
| 용어 | 정의 |
|------|------|
| **Banquet** | 특정 그룹을 위한 공식적, 의례적 만찬 |
| **Banquet Event Order (BEO)** | 호텔이 식음료 행사의 세부사항을 제공하는 양식 |
| **Banquet Round** | 원형 테이블 (60인치=8명, 72인치=10명) |
| **Block** | 그룹을 위해 예약된 객실 수 |
| **Booking** | 시설, 상품, 서비스 사용을 위한 계약 |
| **Break** | 커피, 차 등이 제공되는 세션 간 짧은 휴식 |
| **Breakout Rooms** | 플레너리 세션과 별도로 설치된 소규모 기능실 |
| **Breakout Sessions** | 이벤트 내에서 동시 진행되는 소규모 세션, 패널, 워크숍 |
| **Budget** | 특정 기간 동안의 예상 수입/지출 명세서 |

### C
| 용어 | 정의 |
|------|------|
| **Cancellation Clause** | 계약 취소 시 비취소 당사자에게 지급할 손해액 조항 |
| **Capacity** | 특정 공간에 허용되는 최대 인원 |
| **Catering** | 음식과 음료 제공 |
| **Certificate of Insurance** | 보험 가입 증명 문서 |
| **CMM** | Certificate in Meeting Management (MPI 인증) |
| **CMP** | Certified Meeting Professional |
| **Conference** | 특정 주제에 관한 토론/협의를 위한 참여형 모임 |
| **Congress** | 1) 정보 교환을 위한 정기적 모임 2) 유럽에서 대규모 회의 |
| **Convention** | 공동 목적을 가진 사람들의 모임, 전시회 동반 가능 |
| **CVB** | Convention and Visitors Bureau |

## 호텔/숙박 관련 용어

| 용어 | 정의 |
|------|------|
| **American Plan (AP)** | 객실료에 모든 식사 포함 |
| **European Plan (EP)** | 객실료만, 식사 불포함 |
| **Continental Plan (CP)** | 객실료에 조식 포함 |
| **Modified American Plan (MAP)** | 객실료에 조식+석식 포함 |
| **Comp Room (1/50)** | 50객실당 1객실 무료 제공 정책 |

## 이벤트 유형

| 용어 | 정의 |
|------|------|
| **Association Meeting** | 협회가 주최하는 이벤트 |
| **Board Meeting** | 조직 운영진 회의 |
| **Plenary Session** | 모든 참석자가 참여하는 전체 세션 |
| **Poster Session** | 연구 결과를 포스터로 발표하는 세션 |
| **Workshop** | 실습 중심의 교육 세션 |

## 계약/재무 용어

| 용어 | 정의 |
|------|------|
| **Break-Even Point** | 수입과 지출이 같아지는 지점 |
| **Cash Flow** | 기업 내외로의 자금 이동 |
| **Force Majeure** | 불가항력 조항 (자연재해 등) |
| **RFP** | Request for Proposal (제안요청서) |
| **Bid** | 1) 가격 제시 2) CVB/호텔의 이벤트 유치 제안 |

## 기술 용어

| 용어 | 정의 |
|------|------|
| **A/V** | Audiovisual (음향/영상 장비) |
| **Bandwidth** | 정보 전송 용량 |
| **Webcast** | 인터넷을 통한 실시간 방송 |
| **Videoconference** | 화상 회의 |
