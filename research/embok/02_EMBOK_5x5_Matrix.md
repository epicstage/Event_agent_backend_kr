## 5x5 Matrix 구조

| | ADMINISTRATION | DESIGN | MARKETING | OPERATIONS | RISK |
|---|----------------|--------|-----------|------------|------|
| **1** | Financial Management | Content Management | Marketing Plan Management | Attendee Management | Compliance Management |
| **2** | Human Resources Management | Entertainment Management | Materials Management | Communications Management | Decision Management |
| **3** | Information Management | Environment Management | Merchandise Management | Infrastructure Management | Emergency Management |

## Domain별 상세 설명

### 1. ADMINISTRATION (관리/행정)
자원의 적절한 할당, 지시, 통제를 다룹니다.

| 영역 | 설명 |
|------|------|
| Financial Management | 예산, 회계, 재무 보고 |
| Human Resources Management | 인력 채용, 교육, 관리 |
| Information Management | 정보 수집, 저장, 배포 |
| Procurement Management | 조달, 구매, 계약 |
| Stakeholder Management | 이해관계자 관리 |
| Systems Management | 시스템 및 절차 관리 |
| Time Management | 일정 및 시간 관리 |

### 2. DESIGN (디자인)
이벤트의 콘텐츠, 환경, 경험 설계를 다룹니다.

| 영역 | 설명 |
|------|------|
| Content Management | 콘텐츠 기획 및 개발 |
| Entertainment Management | 엔터테인먼트 요소 |
| Environment Management | 공간 및 환경 디자인 |
| Food & Beverage Management | 케이터링 및 식음료 |
| Production Management | 무대, 조명, 음향 제작 |
| Program Management | 프로그램 구성 |
| Theme Management | 테마 및 브랜딩 |

### 3. MARKETING (마케팅)
이벤트 홍보, 판매, 스폰서십을 다룹니다.

| 영역 | 설명 |
|------|------|
| Marketing Plan Management | 마케팅 전략 및 계획 |
| Materials Management | 홍보물 제작 |
| Merchandise Management | 상품/굿즈 관리 |
| Promotions Management | 프로모션 활동 |
| Public Relations Management | PR 및 언론 대응 |
| Sales Management | 티켓/등록 판매 |
| Sponsorship Management | 스폰서 확보 및 관리 |

### 4. OPERATIONS (운영)
이벤트 현장 운영 및 물류를 다룹니다.

| 영역 | 설명 |
|------|------|
| Attendee Management | 참석자 등록, 체크인 |
| Communications Management | 현장 커뮤니케이션 |
| Infrastructure Management | 시설 및 인프라 |
| Logistics Management | 물류, 운송 |
| Participant Management | 연사, 전시자 관리 |
| Site Management | 장소 관리 |
| Technical Management | 기술 장비 운영 |

### 5. RISK (리스크)
안전, 법률, 위기 관리를 다룹니다.

| 영역 | 설명 |
|------|------|
| Compliance Management | 규정 준수 |
| Decision Management | 의사결정 프로세스 |
| Emergency Management | 비상 상황 대응 |
| Health & Safety Management | 보건 및 안전 |
| Insurance Management | 보험 |
| Legal Management | 계약 및 법적 사항 |
| Security Management | 보안 |

## 에이전트 역할 분담 활용

이 5x5 매트릭스는 Event Agent 시스템에서 역할(R&R)을 나누는 기준으로 활용할 수 있습니다:

```
Agent 1: Administration Agent
  - 예산 관리, 일정 관리, 이해관계자 관리

Agent 2: Design Agent
  - 콘텐츠, 프로그램, 테마, F&B 기획

Agent 3: Marketing Agent
  - 홍보, 판매, 스폰서십, PR

Agent 4: Operations Agent
  - 참석자 관리, 물류, 현장 운영

Agent 5: Risk Agent
  - 안전, 법률, 비상 대응, 보험
```
