## 개요

5개 Knowledge Domain의 상세 분류체계입니다.
각 Domain은 여러 Unit으로 구성되며, 각 Unit은 수십 개의 Topic을 포함합니다.

---

## 1. Administration Domain (관리/행정)

자원의 할당, 지시, 통제를 다루는 영역

### 1.1 Financial Management (재무 관리)

| Unit | Topics |
|------|--------|
| **회계/감사** | Accounting, Auditing, Asset Management |
| **예산** | Bid Preparation, **Budget Development**, Business Plans |
| **현금** | Cash Flow, Cash Handling Procedures, Change Controls |
| **분석** | **Cost/Benefit Analysis**, Cost Controls, Cost Estimating |
| **정책** | Credit Policies, **Deposit Policies**, **Economic Impact** |
| **보고** | Financial Reporting, Fixed/Variable Costs, Foreign Currency |
| **운영** | Inventory Control, Payables & Receivables, Pricing Structures |
| **통제** | Profit Objectives, Purchasing Controls, Rate Negotiation, Resource Definition |

### 1.2 Human Resources Management (인적자원 관리)

| Unit | Topics |
|------|--------|
| **정책** | Behavior Policies, Benefits Management, Conflict Resolution, Discipline |
| **채용** | Employment Regulations, Hiring/Induction, Job Analysis, Job Descriptions |
| **노무** | Labor Relations, Leadership, **Motivation** |
| **조직** | Organizational Structure, Orientation, Paid Staff/Employees |
| **급여** | Payroll Management, Performance Evaluation, Professional Development |
| **인정** | Recognition Programs, Recruitment, Seasonal Staffing |
| **승계** | Succession Planning, Supervision, Team Building |
| **인력** | Temporary/Casual Labor, Temporary Staffing, Termination, Training |
| **기타** | Uniforms, Union Labor, Volunteers |

### 1.3 Information Management (정보 관리)

| Unit | Topics |
|------|--------|
| **커뮤니케이션** | Briefings/Debriefings, Communication Equipment, Communication Planning, Communication Protocols |
| **문서** | Confidentiality Agreements, Database Management, Documentation Procedures, Document Design |
| **평가** | Evaluation/Analysis, Feedback Systems |
| **수집** | Information Acquisition, Information Asset Protection, Information Distribution, Intelligence Gathering |
| **시스템** | Lead Retrieval Systems, Library/Archives, Monitoring & Reporting |
| **미디어** | Presentations, Photography/Videography, Privacy Policies, Record Keeping Procedures |

### 1.4 Procurement Management (조달 관리)

| Unit | Topics |
|------|--------|
| **입찰** | Bid Solicitation, Change Controls |
| **계약** | Contract Management, Performance Evaluation |
| **정책** | Procurement Policies, Purchasing Procedures |
| **품질** | Quality Control, Reimbursement Policies |
| **RFP** | **RFPs/Briefs**, Specifications Definition, Source Definition, Source Selection |

### 1.5 Systems Management (시스템 관리)

| Unit | Topics |
|------|--------|
| **시스템** | Bookkeeping Systems, Change Control Systems, Communication Systems, Database Systems |
| **의사결정** | **Decision Making Systems**, Document Generation, Governance |
| **통합** | **Integration Management**, Inventory Systems, Knowledge Management |
| **운영** | Maintenance Systems, Procedural Manuals, Purchasing Systems |
| **예약** | Reservation/Booking Systems, Routing Systems, Security Systems |

### 1.6 Technology Management (기술 관리)

| Unit | Topics |
|------|--------|
| **장비** | Computers, Digital Electronics, Email & Voice Mail |
| **통신** | Internet/Intranets, Office Equipment, Telecommunications |
| **미디어** | Video, Web-based, Wireless |

### 1.7 Time Management (시간 관리)

| Unit | Topics |
|------|--------|
| **계획** | Activity Definition, Activity Sequencing, Change Controls |
| **관리** | Critical Path, Deadline Management, Duration Estimating |
| **일정** | Float/Slack Time, Lead & Lag Time, Milestone Charts |
| **개발** | Priorities/Sequencing, Recurring Tasks, Schedule Control, Schedule Development, Time Lines |

## 에이전트 역할 분담 적용

이 분류체계를 기반으로 에이전트 역할을 세분화할 수 있습니다:

```
Administration Agent
├── Financial Sub-agent (예산, 회계)
├── HR Sub-agent (인력, 자원봉사)
├── Procurement Sub-agent (조달, 계약)
└── Systems Sub-agent (정보, 기술)

Operations Agent
├── Venue Sub-agent (장소, 시설)
├── Technical Sub-agent (AV, 무대)
├── Logistics Sub-agent (물류, 운송)
└── Registration Sub-agent (등록, 체크인)

Marketing Agent
├── Communications Sub-agent (홍보, PR)
├── Sales Sub-agent (티켓, 등록)
└── Sponsorship Sub-agent (스폰서)

Design Agent
├── Content Sub-agent (프로그램, 세션)
├── Creative Sub-agent (테마, 장식)
└── Production Sub-agent (무대, AV)

Risk Agent
├── Safety Sub-agent (안전, 보건)
├── Legal Sub-agent (계약, 법률)
├── Security Sub-agent (보안)
└── Emergency Sub-agent (비상 대응)
```
